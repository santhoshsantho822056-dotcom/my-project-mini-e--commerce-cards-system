Mini E-Commerce Cards System

Project Description

This is a simple Mini E-Commerce Cards System developed using HTML and CSS. It displays products in attractive cards with images, names, prices, and a Buy Now button.

Features

- Responsive product cards
- Product image display
- Product name and price
- Buy Now button
- Simple and user-friendly design

Technologies Used

- HTML5
- CSS3

Project Structure

project/
│── index.html
│── style.css
│── images/
│── README.md

How to Run

1. Download or clone the project.
2. Open the project folder.
3. Double-click "index.html" or open it in any web browser.
